# Devis.ai

Assistant intelligent pour la génération de devis.

## Structure du projet

- **frontend/** : Application Next.js (Interface Utilisateur)
- **backend/** : API Python FastAPI (Logique métier et IA)

## Déploiement

### Vercel (Frontend)
Assurez-vous de configurer le **Root Directory** sur `frontend` dans les paramètres du projet Vercel.
